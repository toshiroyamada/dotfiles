Toshiro's dotfiles
==================

Install
-------

1.  Clone the repository
    
    `git clone git://github.com/toshiroyamada/dotfiles.git`

2.  Run the install script 

    `./install.sh`

    This will create a symboloic link of dotfiles in the $HOME directory.

3.  Change shell to zsh:

    `chsh -s zsh`

4.  Enjoy!
